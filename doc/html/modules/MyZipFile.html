MyZipFile Module¶

A simple subclass of zipfile.ZipFile that lets you remove leading path element from the zipfile name.

Use exactly as for ZipFile except that on instantiaion you have the argument remove_prefix which is the prefix you want removed from all paths in the zip file, and method writeWithoutPrefix() which will be like ZipFile.write() except that it will remove the prefix
