ThankyouFile Module¶

class ThankyouFile.ThankyouFile(file_path, can_overwrite=False)¶

Bases: AbstractControlFile.AbstractControlFile

Simple class to read/write thank you file. It contains the name of the receipt file that is being acknowledged.

Lines of Thankyou file are:

magic1 basename_for_receipt_file magic2

Note read() and create() are in base class; encode() and decode() are helpers for them.
