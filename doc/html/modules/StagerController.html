StagerController Module¶

class StagerController.StagerController(global_config_path, debug_on=False, oneoff=False)¶

Bases: object

This is the daemon process that manages a group of other class instances which are run in sub-processes, specifically:

  • one LoggerServer

  • one DiskSpaceMonitor for each filesystem

    (via DiskSpaceMonitorLauncher)

  • a DatasetTransferController for each data_stream

  • a DatasetArrivalMonitor for each data_stream which needs it

It is the top-level in MiStaMover (apart from the main program MiStaMover.py which basically just sets a few variables and calls this)

startAll()¶

startDatasetProcs(ds_name)¶

Sets up the data_stream sub-process.

Note that if the source_requires_checksum argument is set to True in the config file then we need to set up two sub-processes:

1. to monitor arrivals, run checksums and produce checksum files (to confirm receipt). 2. to send data on. In other cases we only set up the second sub-process.
