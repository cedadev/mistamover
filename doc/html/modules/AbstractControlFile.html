AbstractControlFile Module¶

class AbstractControlFile.AbstractControlFile(file_path, can_overwrite=False)¶

Bases: object

Simple class to read/write control file.

Note that this base class contains read() and create() that the calling code will actually call, but they rely on the subclass to provide certain helper functions.

In particular, subclasses must provide:
magic1, magic2: strings to put at start and end of file encode: method to turn input data into lines for the file decode: method to turn lines from the file into data
create(*args, **kwargs)¶

Create the file from input data

read()¶

Read the file, return decoded contents

readlines()¶

Read lines excluding (but checking) the ‘magic’ lines at start and end

stripNewline(line)¶
writeline(data)¶
writelines(data)¶

Write lines, adding on the ‘magic’ lines at start and end.
