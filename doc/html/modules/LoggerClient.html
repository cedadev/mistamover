LoggerClient Module¶

class LoggerClient.LoggerClient(config, host='localhost', port=None, tag='MiStaMover', name=None, debug_on=False)¶

Bases: object

Class that connects to a running LoggerServer, and also sends email for messages which exceed a certain level.

(The sending email could perhaps be moved to the server - it is possible to test the level in the server - but this way the app can decide whether to send email by specifying send_email=True/False on logMessage(), overriding the config-based decision.)
